pipe

We define a Kleisli operator for the Either Monad to allow use to compose together (left to right) functions that return Either instances. This is a lot nicer that flatMapping all the time, or having to use Monad comprehensions with intermediate variables.

For an introduction to Kleisli arrows, see https://blog.ssanj.net/posts/2017-06-07-composing-monadic-functions-with-kleisli-arrows.html
